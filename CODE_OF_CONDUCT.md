# 帝國海軍 行動規範

## 軍人勅諭ニ基ヅク五箇條ノ訓戒

大元帥陛下（[@imperial-household](https://github.com/imperial-household) Organization Owner）ノ聖旨ヲ奉戴シ、本組織ニ參加スル一切ノ者ハ以下ノ五箇條ヲ嚴守スベシ。

---

### 第壹條 忠節（チュウセツ）

本組織ノ構成員ハ大元帥陛下及ビ帝國海軍ニ對シ絶對ノ忠節ヲ盡スベシ。
組織ノ方針及ビ上官ノ指示ニ從ヒ、私情ヲ挾マズ職務ヲ遂行スルコト。
リポジトリノ目的ニ反スル行爲、組織ノ秩序ヲ亂ス行爲ハ忠節ニ悖ルモノトシテ嚴ニ之ヲ禁ズ。

帝國海軍ハ GitHub ナル公海上ニ錨泊スル以上、其ノ港灣規則（[GitHub Community Guidelines](https://docs.github.com/en/site-policy/github-terms/github-community-guidelines)）ヲ遵守スルハ當然ノ義務ナリ。寄港地ノ法規ヲ蔑ロニスル艦船ハ入港ヲ拒否サレ、帝國ノ威信ヲ損ナフ。故ニ GitHub ノ定ムル諸規定ヘノ服從ハ忠節ノ一部ト心得ベシ。

### 第貳條 禮儀（レイギ）

構成員相互ノ間ニ於テハ常ニ禮儀ヲ正シクシ、上下ノ分ヲ辨ヘ、言語ヲ愼ムベシ。
Issue、Pull Request、Discussion 等ニ於ケル發言ハ丁重ヲ旨トシ、他ノ構成員ヲ侮辱シ又ハ誹謗スル言辭ハ一切之ヲ許サズ。

帝國海軍ハ萬國ヨリ將兵ヲ迎フルコトヲ以テ光榮トス。人種、民族、國籍、信條、性別、性的指向、障害ノ有無其ノ他一切ノ屬性ニ依ル差別的言動ハ皇軍ノ品位ヲ汚スモノニシテ嚴ニ之ヲ禁ズ。嫌ガラセ（Harassment）、脅迫、個人攻擊ノ類ハ帝國軍人タル者ノ爲ス所ニ非ズ。斯カル卑劣ナル行爲ハ敵ニ弱點ヲ曝スニ等シク、發覺セシ場合ハ第三種處分（永久追放）ノ對象トス。

英國海軍ノ King's Regulations ニ曰ク「紳士タレ（Be a gentleman）」ト。帝國海軍モ亦然リ。

### 第參條 武勇（ブユウ）

技術的困難ニ遭遇セシ際ハ怯懦ナル態度ヲ取ルコトナク、果敢ニ之ニ立向フベシ。
バグ報告、コードレビュー、新機能ノ提案等ニ於テハ積極果敢ナル姿勢ヲ以テ臨ムコト。
但シ武勇ハ蠻勇ニ非ズ。無謀ナル變更ヲ强行シ他ノ成果ヲ破壞スル行爲ハ嚴ニ戒ムベシ。

### 第肆條 信義（シンギ）

構成員ハ信義ヲ重ンジ、虛僞ノ報告ヲ爲サザルコト。
コミットメッセージ、ドキュメント、Issue 等ニ於テハ事實ヲ正確ニ記載スベシ。
他者ノ成果ヲ剽竊シ自己ノ功績ト爲ス行爲ハ信義ニ悖リ、嚴罰ニ處ス。
他者ノ個人情報ヲ無斷ニテ公開スル行爲モ亦信義ニ反ス。味方ノ位置ヲ敵ニ漏ラスニ等シキ利敵行爲ナリ。

### 第伍條 質素（シッソ）

構成員ハ質素ヲ旨トシ、華美ニ流レザルコト。
コードハ簡潔明瞭ヲ以テ良シトシ、不要ナル複雜化ヲ避クベシ。
リポジトリノ容量ヲ徒ニ增大セシムル行爲ハ質素ノ德ニ反スルモノトス。

## 違反者ノ處分

上記五箇條ニ違反セシ者ハ、其ノ情狀ニ應ジ左ノ處分ヲ行フ。

| 處分等級 | 處分内容 | 適用例 |
|---|---|---|
| 第一種（訓戒） | 警告ノ發出 | 輕微ナル禮儀違反、初犯 |
| 第二種（謹愼） | 一定期間ノ活動停止 | 反復セル規律違反 |
| 第三種（除隊） | 組織ヨリ永久追放（Ban） | 重大ナル信義違反、破壞行爲、嫌ガラセ |

處分ノ決定ハ海軍省（Organization管理者）ニ於テ軍法會議ニ準ジタル審議ヲ經テ之ヲ行フ。
處分ニ對スル不服ハ海軍省軍務局ニ抗告スルコトヲ得。軍務局ハ之ヲ審理シ、猶ホ解決ヲ見ザル場合ハ海軍大臣ヨリ大元帥陛下（[@imperial-household](https://github.com/imperial-household) Organization Owner）ニ上奏スルモノトス。

尚、處分ノ手續ニ依ラズ [GitHub ノ港灣當局](https://support.github.com/contact/report-abuse)ニ直接通報スルコトモ妨ゲズ。軍ノ手續ヲ經ズトモ寄港地ノ警察權ハ常ニ有效ナリ。
